#### 预览

#### 食用方法

下载Source，把Saka文件夹放入blog目录。

修改 `config.yml` 中`site:theme`字段与`build:copy`字段如下：

```
site:
    ...
    comment:  # 这里填写你的多说域名short-name
    theme: Saka
    ...
build:
    ...
    copy:
        - source/images
    ...
```
